# The Golf Hub

A private, iPhone-first golf app — prototype / demo.

One home for a single golfer's game: strokes-gained vs a 5-handicap, a par-baseline
scoring graph, per-hole round tracker with a course book, club yardages (My Bag),
a caddie view with live Kansas City weather, and an improvement plan.

**Live demo:** https://1browneric.github.io/golf-hub/

Status: work-in-progress design prototype. Round/season figures are sample
placeholders; club yardages are real. Not affiliated with any brand shown.
